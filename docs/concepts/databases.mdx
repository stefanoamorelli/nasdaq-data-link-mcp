## What are Databases?

Databases are collections of related datasets. Each database focuses on a specific domain like economics, equities, or funds.

## Featured Databases

### World Bank (WORLDBANK)

Global development indicators covering 200+ countries.

<Card title="Coverage" icon="globe">
  - GDP, Population, Inflation
  - CO₂ emissions, Energy use
  - Education, Health indicators
  - Trade, Investment data
</Card>

**Example Datasets:**
- `WORLDBANK/USA_NY.GDP.MKTP.CD` - US GDP
- `WORLDBANK/ITA_SP.POP.TOTL` - Italy population
- `WORLDBANK/DEU_EN.ATM.CO2E.PC` - Germany CO₂ per capita

---

### Equities 360 (QOR)

Comprehensive company fundamental data and statistics.

<Card title="Coverage" icon="building">
  - Income statements
  - Balance sheets
  - Cash flow statements
  - Valuation metrics
  - Company statistics
</Card>

**Example Datasets:**
- `QOR/STATS_MSFT` - Microsoft statistics
- `QOR/FS_AAPL` - Apple fundamentals
- `QOR/BS_GOOGL` - Google balance sheet

---

### Fund Network (NFN)

Mutual fund and ETF information.

<Card title="Coverage" icon="chart-line">

  - Share class information

  - Fees and expenses
</Card>

**Example Datasets:**
- `NFN/MFRFM` - Fund master report

### Retail Trading Activity (NDAQ/RTAT)

Retail investor trading patterns.

<Card title="Coverage" icon="users">

**Example Datasets:**

## Discovering Databases

### Using list_databases

```python
# Ask Claude:
"List all available databases"

# Claude calls:
list_databases()
```

Returns all databases with descriptions.

### Using search_datasets

```python
# Ask Claude:
"Find databases about economics"

# Claude calls:
search_datasets(query="economics")
```

Searches across all databases for relevant datasets.

## Database Structure

Each database contains multiple datasets:

```
WORLDBANK/
├── USA_NY.GDP.MKTP.CD (US GDP)
├── ITA_NY.GDP.MKTP.CD (Italy GDP)
├── DEU_NY.GDP.MKTP.CD (Germany GDP)
└── ... (thousands more)

QOR/
├── STATS_AAPL (Apple statistics)
├── STATS_MSFT (Microsoft statistics)
├── FS_AAPL (Apple fundamentals)
└── ... (thousands more)
```

## Database Access

All databases are accessible through the same 5 tools:

1. **search_datasets** - Find datasets in any database
2. **get_dataset** - Get data from any database
3. **get_dataset_metadata** - Inspect structure
4. **list_databases** - Browse all databases
5. **export_dataset** - Export from any database
