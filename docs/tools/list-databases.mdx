description: 'List all available databases on Nasdaq Data Link'
---

## Overview

Returns a list of all databases available on Nasdaq Data Link with their descriptions.

## Parameters

No parameters required.

## Response

```json
[
  {
    "code": "WORLDBANK",
    "name": "World Bank",
    "description": "Global development indicators"
  },
  ...
]
```

## Example

```python
list_databases()
```
