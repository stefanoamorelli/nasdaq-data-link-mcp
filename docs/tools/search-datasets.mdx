## Overview

Search for datasets across all Nasdaq Data Link databases using keywords. Returns matching datasets with their codes, names, and descriptions.

## Parameters

<ParamField path="query" type="string" required>
  Search term to find relevant datasets. Can be:
  - Keywords (e.g., "GDP", "inflation")
  - Database names (e.g., "WORLDBANK")
  - Ticker symbols (e.g., "AAPL", "MSFT")
  - Topics (e.g., "oil prices", "mutual funds")
</ParamField>

## Response

Returns a list of matching datasets:

```json
[
  {
    "code": "WORLDBANK/USA_NY.GDP.MKTP.CD",
    "name": "United States GDP (current US$)",
    "description": "GDP at purchaser's prices is the sum..."
  },
  ...
]
```

## Best Practices

<Steps>
  <Step title="Start Broad">
    Begin with general keywords like "GDP" or "stocks"
  </Step>

  <Step title="Refine Results">
    Add more specific terms: "GDP Italy" or "tech stocks"
  </Step>

  <Step title="Check Descriptions">
    Read dataset descriptions to find the right match
  </Step>

  <Step title="Use Dataset Codes">
    Once you find the right dataset, use `get_dataset` with its code
  </Step>
</Steps>

## Tips

<Tip>
  Search returns up to 10 results. Be specific to get better matches.
</Tip>

<Tip>
  Database codes can be used as search terms. Try "QOR" for Equities 360 data.
</Tip>

<Warning>
  Search looks for exact word matches. "GDP" finds more than "gross domestic product"
</Warning>
