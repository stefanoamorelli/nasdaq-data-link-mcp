## What is Nasdaq Data Link MCP?

Nasdaq Data Link MCP is a community-maintained [Model Context Protocol (MCP)](https://modelcontextprotocol.io) server that enables large language models like Claude to access and analyze data from [Nasdaq Data Link](https://data.nasdaq.com/).

## Why MCP?

The Model Context Protocol allows AI assistants to safely access external data sources. With this server:
- Claude can query real financial data on your behalf
- All data stays secure in your local environment
- No manual API calls needed - just ask questions
